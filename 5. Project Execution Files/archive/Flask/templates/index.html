About Image Caption Generation

An image caption description is a written caption text that explains the important details of a picture.It involves generating a human readable textual description given an image,such as a photograph. For human it is very simple task.but for a computer it is extremely difficult since it requires both understanding the content of an image and how to translate this understanding into natural language.

Recently, deep learninf methods have displaced classical methods and are achieving state-of-the-art rresults for the problem of automatically generating descriptions,called "captions",for images. Applications are as follows:

  • Image indexing
  • for visually impaired persons
  • For social media
  • Usage in virtual assistants
  • Several other natural language processing applications.
